Quick Start

Get up and running with BrandMention Alert API in 5 minutes.

1. Get Your API Key

Sign up for a free account and get your API key from the dashboard.

Get Free API Key

2. Make Your First Request

Use this cURL command to test your API key:

curl -X GET "https://api.brandmentalert.com/v1/search?keyword=OpenAI&limit=5" \
  -H "X-API-Key: your-api-key-here"

3. Integrate into Your App

Check the JavaScript examples below to integrate into your application.

Authentication

All API requests require authentication using your API key.

HEADER X-API-Key: your-api-key

Getting Your API Key

After signing up, you'll find your API key in the dashboard. Keep it secure and never expose it in client-side code.

Important: Always use HTTPS and keep your API key confidential.

Rate Limiting

API requests are limited to ensure fair usage and system stability.

Free Plan

Requests per minute: 60
Requests per day: 1,000
Monthly searches: 100

Pro Plan

Requests per minute: 300
Requests per day: 10,000
Monthly searches: 5,000

Business Plan

Requests per minute: 1,000
Requests per day: 50,000
Monthly searches: 20,000

Rate Limit Headers

Every API response includes rate limit headers:

Header Description
X-RateLimit-Limit Total requests allowed in current period
X-RateLimit-Remaining Remaining requests in current period
X-RateLimit-Reset Unix timestamp when limits reset

Handling Rate Limits

When you exceed rate limits, the API returns a 429 Too Many Requests status code.

// Example rate limit response
{
  "error": {
    "code": 429,
    "message": "Rate limit exceeded",
    "retry_after": 30
  }
}

Search Mentions

Search for brand mentions across all supported platforms.

GET /v1/search

Parameters

Parameter Type Required Description
keyword string Yes Brand name or keyword to search for
limit number No Number of results (1-50, default: 10)
platform string No Platform filter: all, reddit, twitter, news
timeframe string No Time range: 1h, 24h, 7d, 30d, all

Example Request

// Search for OpenAI mentions on Reddit
GET /v1/search?keyword=OpenAI&platform=reddit&limit=5&timeframe=7d
Headers:

Webhook Events

Event Type Description
new_mention Triggered when a new mention is found
high_engagement Triggered when a mention gets high engagement
sentiment_alert Triggered for negative sentiment mentions

Webhook Security

Each webhook request includes a signature header for verification:

X-Webhook-Signature: sha256=abc123def456...

Verify the signature using your webhook secret to ensure the request is from BrandMention Alert.

Retries and Failures

Webhooks are retried with exponential backoff for up to 24 hours if your endpoint returns non-2xx status codes.

JavaScript Examples

Search Mentions

const searchMentions = async (keyword, limit = 10) => {
  try {
    const response = await fetch(
      `https://api.brandmentalert.com/v1/search?keyword=${encodeURIComponent(keyword)}&limit=${limit}`,
      {
        headers: {
          'X-API-Key': 'your-api-key-here',
          'Content-Type': 'application/json'
        }
      }
    );
    
    if (!response.ok) {
      throw new Error(`HTTP error! status: ${response.status}`);
    }
    
    const data = await response.json();
    return data;
  } catch (error) {
    console.error('Error searching mentions:', error);
    throw error;
  }
};

// Usage example
searchMentions('OpenAI', 5)
  .then(data => {
    console.log('Found mentions:', data.data.mentions);
    console.log('Summary:', data.data.summary);
  })
  .catch(error => console.error('Error:', error));

Handle Rate Limits

const apiRequestWithRetry = async (url, options, retries = 3) => {
  try {
    const response = await fetch(url, options);
    
    // Check for rate limit
    if (response.status === 429) {
      const retryAfter = response.headers.get('Retry-After') || 1;
      console.log(`Rate limited. Retrying after ${retryAfter} seconds...`);
      
      if (retries > 0) {
        await new Promise(resolve => setTimeout(resolve, retryAfter * 1000));
        return apiRequestWithRetry(url, options, retries - 1);
      }
    }
    
    if (!response.ok) {
      throw new Error(`HTTP error! status: ${response.status}`);
    }
    
    return await response.json();
  } catch (error) {
    console.error('API request failed:', error);
    throw error;
  }
};

// Usage with rate limit handling
apiRequestWithRetry(
  'https://api.brandmentalert.com/v1/search?keyword=OpenAI&limit=5',
  {
    headers: {
      'X-API-Key': 'your-api-key-here'
    }
  }
).then(data => console.log(data));
