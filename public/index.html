Search Mentions

Search for brand mentions across platforms.

GET /api/search?keyword=BRAND&limit=10

Parameters

Parameter Type Description
keyword string Brand name to search for (required)
limit number Number of results (1-50, default: 10)
platform string Platform to search: all, reddit, news

Example Request

curl -X GET "http://localhost:3000/api/search?keyword=OpenAI&limit=5" \
  -H "X-API-Key: your-api-key"

Authentication

All API requests require authentication using your API key.

HEADER X-API-Key: your-api-key

Getting Your API Key

After signing up, you'll find your API key in the dashboard. Keep it secure and never expose it in client-side code.

Important: Always use HTTPS and keep your API key confidential.
